# Source Tree Map

This page maps each top level folder to a responsibility.

## Top Level

```text
src/common
src/dashjs
src/hlsjs
src/videojs
src/index-web.js
src/index-hbbtv.js
```

## Responsibilities

- common: shared abstractions, events, trackers, utility modules
- videojs|dashjs|hlsjs: core specific playback implementations
- index-web.js: web entrypoint
- index-hbbtv.js: hbbtv entrypoint
