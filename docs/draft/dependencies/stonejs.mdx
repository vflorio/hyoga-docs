# StoneJS

StoneJS is the API client used to fetch content metadata and playback related data.

## Role In Hyoga

- Provides data used to resolve playable sources
- Carries environment specific values like endpoint and realm
- Can include token based access for restricted content

## Typical Bootstrap

```js
const stone = new StoneJS({
  endpoint: "https://eu2-prod.disco-api.com",
  realm: "dplayit",
  deviceId: "SmartTV:Tizen-4.0:dplay:3.1.0",
  // token: "..."
});
```
