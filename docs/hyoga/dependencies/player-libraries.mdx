# Player Libraries

Hyoga wraps multiple core players with one abstraction layer.

## Supported Cores

- videojs
- dashjs
- hlsjs

## Source Mapping

```text
src/videojs/player-core.js
src/dashjs/player-core.js
src/hlsjs/player-core.js
```

## Related Runtime Assets

```text
src/videojs/lib/*
src/dashjs/lib/*
src/hlsjs/lib/*
```
