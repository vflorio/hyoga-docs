# React Playground

import HyogaPlayground from '@site/src/components/HyogaPlayground';

This page renders a real Hyoga runtime instance in React using:

- Bowser script (required by Hyoga runtime)
- Hyoga web bundle script (CDN or local URL)
- Custom elements `hyoga-player` and `hyoga-videoplayer`

No trackers, gtag, or ads setup is required for this minimal demo.

## Presets Included

- Video preset based on `public.aurora.enhanced.live` with `assetid=18322`
- Channel preset based on `stage-public.aurora.enhanced.live` with `assetid=6` and source params

## Minimal Runtime Inputs

The component uses the minimal attribute set used in runtime integrations:

- `id`, `uid`, `playerselector`
- `assetid`, `playbacktype`, `sourcetype`
- `videolibrary`, `endpoint`, `realm`, `locale`
- `autoplay`, `muted`

## Script Source

Default Hyoga script URL is:

```bash
https://cdn.hyogaplayer.com/hyoga-web.min.js
```

For local debug you can point `Hyoga script URL` to:

```bash
http://localhost:5173/hyoga-web.min.js
```

or to your CDN build URL.
