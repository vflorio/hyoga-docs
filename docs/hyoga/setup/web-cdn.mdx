# Setup Web Via CDN

Hyoga package is not published on npm. Use CDN or internal static hosting.

## Minimal Script Include

```html
<script src="https://your-cdn.example.com/hyoga-web.min.js"></script>
```

## Minimal Host Page Requirements

- Root container for your app
- Runtime global config (if needed)
- Player and tracker bootstrap order
